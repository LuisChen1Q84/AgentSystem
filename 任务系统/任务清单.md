## 任务格式规范

每个任务必须包含以下字段：

```
- [ ] 任务名称
  - 创建时间：YYYY-MM-DD HH:MM
  - 截止时间：YYYY-MM-DD（可选）
  - 来源：会话/用户输入/系统
  - 优先级：紧急重要/重要不紧急/日常事项
  - 状态：待办/进行中/已完成
  - 备注：...
```

## 自动追踪规则

### 时间意图捕获

当用户输入包含以下词汇时，自动创建任务：

| 触发词 | 含义 | 自动添加 |
|--------|------|----------|
| 明天 | 截止=明天 | 紧急重要 |
| 后天 | 截止=后天 | 重要不紧急 |
| 下周 | 截止=下周 | 重要不紧急 |
| 月X日 | 截止=指定日期 | 根据日期计算 |
| 记得 | 提醒=是 | 重要不紧急 |
| 别忘了 | 提醒=是 | 重要不紧急 |
| 计划 | 提醒=是 | 日常事项 |
| 打算 | 提醒=是 | 日常事项 |

### 自动添加流程

```
用户输入 → 检测时间意图 → 解析任务内容
     ↓
判断优先级 → 创建任务 → 更新任务清单
     ↓
告知用户 → 晨间简报显示
```

### 发布状态自动更新

当用户说"已发布"、"发布了"，自动：
1. 查找相关的待发布任务
2. 标记为已完成
3. 移动到归档

### 每日检查

每次会话开始时：
1. 检查今日待办
2. 检查已过期任务
3. 在晨间简报中提醒
