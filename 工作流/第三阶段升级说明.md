# 第三阶段升级说明（智能决策与经营看板）

## 目标

将系统从“执行工具”升级为“经营驾驶舱”。

## 已落地能力

1. 经营看板
- 脚本：`scripts/business_dashboard.py`
- 产出：`日志/经营看板/YYYY-MM-DD.md`

2. 风险雷达
- 脚本：`scripts/risk_radar.py`
- 产出：`日志/风险雷达/YYYY-MM-DD.md`
- 风险维度：过期任务、错误率、低置信文档

3. 每周复盘
- 脚本：`scripts/weekly_review.py`
- 产出：`日志/每周复盘/<range>.md`

4. 轻量OKR
- 脚本：`scripts/okr_tracker.py`
- 数据：`目标系统/okr.json`
- 产出：`日志/OKR/YYYY-MM-DD.md`

5. 经营节奏编排
- 日节奏：`make cycle-daily`
- 周节奏：`make cycle-weekly`
- 月节奏：`make cycle-monthly`

## 常用命令

```bash
make risk
make dashboard
make weekly-review
make okr-init
make okr-report
make cycle-daily
make cycle-weekly
make cycle-monthly
```
