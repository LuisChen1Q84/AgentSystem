# 每周存储复盘

**版本**: v1.0
**说明**: 每周自动执行的文件存储检查和优化

---

## 触发条件

- 每周一晨间简报时
- 用户说"存储复盘"
- 用户说"/storage-review"

---

## 执行流程

### 1. 检查知识库结构

```
检查各知识库目录完整性：
- 知识库/important_document/
- 知识库/hankun_information/
- 知识库/pbc_weixin/
- 知识库/regulation_weixin/
- 知识库/business_weixin/
- 知识库/agreement_weixin/
- 知识库/iresearch_information/
```

### 2. 错位文件检查

检查是否有文件放错位置：

```
预期路径 vs 实际路径对比
- 政策简报 → /Volumes/Luis_MacData/Documents/trabajo/
- 情景记忆 → 记忆库/情景记忆/
- 语义记忆 → 记忆库/语义记忆/
- 日记 → 日记/
```

### 3. 空目录检查

```
find /Volumes/Luis_MacData/AgentSystem -type d -empty
```

### 4. 文件统计

统计各类文件数量：
- 情景记忆文件数
- 语义记忆文件数
- 强制规则数
- 技能文件数
- 知识库文件总数

### 5. 生成报告

输出格式：

```
=== 每周存储复盘 | 2026-02-24 ===

📁 知识库检查：
✓ important_document: 35 files
✓ hankun_information: 80 files
✓ pbc_weixin: 73 files
✓ regulation_weixin: 105 files
✓ business_weixin: 31 files
✓ agreement_weixin: 45 files
✓ iresearch_information: 38 files

📊 文件统计：
- 情景记忆: X files
- 语义记忆: X files
- 强制规则: X files
- 总计: X files

🔍 错位文件检查：
- 错位文件: 0
- 空目录: 0

📈 存储路由准确率：100%

⚠️ 需要调整：
- 无

✅ 系统状态：正常
```

---

## 自动执行

在CLAUDE.md中添加规则：

```
每周一晨间简报时自动执行：
1. 运行每周存储复盘流程
2. 生成复盘报告（日志/YYYY-MM-DD_每周存储复盘.md）
3. 在简报中显示结果
4. 如发现问题，提示用户
```
