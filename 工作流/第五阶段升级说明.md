# 第五阶段升级说明（智能闭环与策略自调优）

## 目标

把系统从“能执行”升级为“会判断、会复盘、会自调优”。

## 已落地能力

1. 决策引擎
- 脚本：`scripts/decision_engine.py`
- 产出：`日志/决策/YYYY-MM-DD.md`
- 输入：任务状态、风险级别、运营指标、OKR进度
- 输出：按分值排序的优先动作列表

2. 闭环优化器
- 脚本：`scripts/closed_loop_optimizer.py`
- 产出：`日志/闭环优化/YYYY-MM-DD.md`
- 策略文件：`目标系统/optimization_policy.json`
- 能力：根据近14天日志错误率、完成率、待办规模自动建议重试参数/WIP上限/执行焦点

3. 战略简报
- 脚本：`scripts/strategy_brief.py`
- 产出：`日志/战略简报/YYYY-MM-DD.md`
- 输入：经营看板、风险雷达、指标报告、决策清单、闭环优化报告

4. 智能节奏编排
- 新增命令：`make cycle-intel`
- 执行链路：`metrics -> risk -> decision -> optimize -> strategy`

## 常用命令

```bash
make decision
make optimize
make strategy
make cycle-intel
```

## 价值

- 让系统从“报表驱动”升级为“动作驱动”
- 让优化不依赖人工记忆，而是持续形成策略文件
- 为 1 人公司提供可执行的经营节奏和优先级秩序
