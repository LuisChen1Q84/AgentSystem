# 第六阶段升级说明（预测驱动与自学习进化）

## 目标

把系统从“会判断”升级为“会预判、会实验、会学习”。

## 已落地能力

1. 经营预测
- 脚本：`scripts/forecast_engine.py`
- 产出：`日志/预测/YYYY-MM-DD.md`
- 能力：基于近30天指标做短期趋势预测与风险预警

2. 实验管理
- 脚本：`scripts/experiment_manager.py`
- 产出：`日志/实验/YYYY-MM-DD.md`
- 数据：`目标系统/experiments.json`
- 能力：根据决策Top动作和策略焦点自动生成7天实验

3. 学习闭环
- 脚本：`scripts/learning_loop.py`
- 产出：`记忆库/语义记忆/经营学习卡.md`
- 能力：从战略简报/决策/预测中提炼高价值信号并沉淀

4. 进化节奏编排
- 新增命令：`make cycle-evolve`
- 执行链路：`cycle-intel -> weekly-review -> dashboard`

## 常用命令

```bash
make forecast
make experiment
make learning
make cycle-evolve
```

## 价值

- 让经营动作具备前瞻性，而非事后响应
- 让改进以实验形式持续可追踪
- 让系统持续形成可复用的经营认知
