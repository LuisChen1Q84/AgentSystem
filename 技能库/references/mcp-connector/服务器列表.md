# MCP 服务器列表

## 官方推荐服务器

### 文件系统 (filesystem)
- **包**: `@modelcontextprotocol/server-filesystem`
- **用途**: 读取、写入、遍历本地文件
- **适用场景**: 
  - 读取用户上传的文档
  - 批量处理文件
  - 自动生成报告文件

### 网络请求 (fetch)
- **包**: `@modelcontextprotocol/server-fetch`
- **用途**: HTTP/HTTPS请求
- **适用场景**:
  - 获取政策原文
  - 调用外部API
  - 网页内容抓取

### SQLite (sqlite)
- **包**: `@modelcontextprotocol/server-sqlite`
- **用途**: SQLite数据库操作
- **适用场景**:
  - 查询业务数据
  - 数据分析
  - 报表生成

### GitHub (github)
- **包**: `@modelcontextprotocol/server-github`
- **用途**: GitHub API操作
- **适用场景**:
  - 代码仓库分析
  - 自动创建Issue/PR
  - 代码搜索

### Brave搜索 (brave-search)
- **包**: `@modelcontextprotocol/server-brave-search`
- **用途**: 网络搜索
- **适用场景**:
  - 政策文件搜索
  - 行业新闻检索
  - 竞品分析

### 顺序思考 (sequential-thinking)
- **包**: `@modelcontextprotocol/server-sequential-thinking`
- **用途**: 结构化思维链
- **适用场景**:
  - 复杂问题分析
  - 多步骤推理
  - 决策辅助

## 推荐组合

### 政策分析工作流
```
fetch (获取政策原文) 
  → filesystem (保存到知识库)
  → sqlite (记录元数据)
  → sequential-thinking (深度分析)
```

### 数据报告工作流
```
sqlite (查询业务数据)
  → filesystem (生成Excel报告)
  → github (上传到仓库)
```

### 竞品监控工作流
```
brave-search (搜索竞品动态)
  → fetch (获取详情)
  → filesystem (保存监控报告)
```
