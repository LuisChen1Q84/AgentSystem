# 信息策展规则

## 过滤规则

### 排除内容
- 广告、软文、纯推广内容
- 标题党、博眼球内容
- 重复/抄袭内容

### 保留内容
- 深度分析、独家新闻
- 技术干货、行业洞察
- 原创研究、数据报告

## 优先级排序

1. **行业重大政策变动** - 金融监管政策、行业法规
2. **技术创新突破** - 新技术、新工具
3. **市场趋势分析** - 行业报告、市场分析
4. **竞争动态** - 竞品消息、行业合作

## 来源权重

| 来源 | 权重 | 说明 |
|------|------|------|
| HackerNews | 1.2 | 技术社区，质量较高 |
| GitHub | 1.2 | 开源项目，实用性强 |
| Reddit | 1.0 | 社区讨论 |
| RSS | 1.0 | 博客新闻 |
| Web Search | 0.8 | 通用搜索 |

## 关键词匹配

### 高权重关键词
- 政策、监管、央行、金融
- AI、LLM、模型、算法
- 支付、清算、数字货币
- 创新、突破、首发

### 低权重/排除关键词
- 广告、推广、优惠
- 标题党、震惊
