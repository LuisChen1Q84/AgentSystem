# 语义记忆：policy-pbc工作流模式

## 概述

记录policy-pbc系统的标准化工作流程和核心方法论

## 输入格式

```
政策主题：[政策名称/关键词]
目标地域：[省/市/区，可选]
分析重点：[可选，特定领域]
```

## 四阶段工作流

1. **情报挖掘**：搜索 + 知识库检索 + 材料整理
2. **专业分析**：政策协同性评估 + 支付行业分析 + 地域温差解读
3. **事实核查**：数据核查 + 来源标注 + 审慎判断
4. **格式化交付**：自检 + 输出简报 + 文件生成

## 核心原则

- 结论优先
- 叙述为主
- 客观研判（不写建议）
- 数据支撑

## 来源置信度

- L1：央行原文
- L2：汉坤解读
- L3：媒体报道
- L4：社交媒体（仅参考）
