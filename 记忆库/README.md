# 记忆库说明

## 三层记忆架构

本目录采用三层记忆架构：

1. **情景记忆**：具体事件和对话，按月归档
2. **语义记忆**：提炼的知识和模式，可复用
3. **强制规则**：系统行为约束，必须遵守

## 目录结构

```
记忆库/
├── README.md           # 本文件
├── 情景记忆/           # 具体事件（按月归档）
│   └── 2026-02/
├── 语义记忆/           # 提炼知识
└── 强制规则/           # 系统规则
```

## 命名规范

- 情景记忆：MMDD-简短描述.md
- 语义记忆：[主题]_[类型].md
- 强制规则：强制规则_[领域].md

## 检索优先级

1. 先读本README
2. 根据任务类型选择对应子目录
3. 使用关键字搜索语义记忆
