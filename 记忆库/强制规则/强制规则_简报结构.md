# 强制规则：简报结构

## 输出简报标准结构

### 核心原则

1. 结论优先
2. 叙述为主（少用表格）
3. 客观研判（不写"建议"）
4. 精炼聚焦
5. 数据支撑

### 标准结构

```
# [政策/项目名称] 监管影响深度洞察

**报告日期**：YYYY-MM-DD

---

## 核心结论

[用1-2段话概括全文核心要点]

---

## 展开解读

### 洞察一：[洞察标题]

**结论**：[一句话概括核心洞察]

**支撑分析**：
- [必要的政策内容/数据]

**深层洞察**：
- [基于分析得出的更深层次判断]

---

**报告编制**：政策决策参谋
**方法论说明**：推断性内容已标注"我们判断"、"我们预计"等限定词
```

### 来源标注

- L1：央行官网、监管文件原文
- L2：汉坤等专业律所解读
- L3：主流媒体财经报道
- L4：社交媒体（仅作参考）
