# 强制规则：智能存储路由

## 存储规则

每次创建文件时自动执行：
1. 识别内容类型
2. 查询存储规则
3. 构建完整文件路径
4. 验证路径合规性
5. 执行存储操作
6. 告知用户保存位置和决策理由
