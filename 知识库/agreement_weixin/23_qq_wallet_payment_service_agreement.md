## 一、协议基本信息

**签订方**：财付通支付科技有限公司（以下简称"财付通"或"本公司"）与商户

**关联协议**：《财付通服务协议》、《QQ钱包支付商户平台使用协议》

---

## 二、核心定义

| 术语 | 定义 |
|------|------|
| 手机QQ | 腾讯提供的跨平台通讯工具 |
| 手机QQ钱包支付 | 本公司依托手机QQ平台提供的第三方支付软件系统和服务 |
| 财付通商户号 | 用于记载商户身份信息、交易信息、资金余额的电子簿记 |
| 商户平台 | 本公司提供和维护的商户管理软件系统 |
| 商户平台登录账号 | 商户凭以登录商户平台，操作财付通商户号的账号 |

---

## 三、服务内容

**支付场景**（第2.1条）：
1. 公众号/H5支付
2. APP支付
3. 扫码支付（包含PC扫码）
4. 付款码支付

**服务限制**：
- 只能用于在财付通平台申报并经核准的商品或服务
- 每笔订单不能超过本公司规定的限额
- 本公司有权根据商户经营风险调整交易额度

---

## 四、本公司（财付通）的权利和义务

**主要权利**（第3条）：
1. 负责支付系统的建设、运行和管理，确保安全性
2. 提供交易查询系统及查询接口
3. 提供"客户证书"服务
4. 定期或不定期审核商户业务类型
5. 对风险等级较高的商户采取调整交易限额、延迟结算、现场检查等措施
6. 账户无交易记录超过一定期限有权暂停服务
7. 有权直接从商户财付通商户号扣取相应金额（违规费用、赔偿等）
8. 对可疑交易进行延迟结算
9. 进行商户巡检（现场巡检或远程巡检）

---

## 五、商户权利和义务

**主要义务**（第4条）：

| 类别 | 具体要求 |
|------|----------|
| 资质提供 | 如实提供营业执照、税务登记证、组织机构代码证、法定代表人身份证明等 |
| 账户安全 | 不得泄露、转让商户号、登录账号、接口技术、安全协议及证书 |
| 敏感信息 | 不得存储用户银行账户密码、磁道信息、银行卡验证码等敏感信息 |
| 徽标使用 | 可展示"QQ钱包"logo和标识，但需与本公司提供样本一致 |
| 推广义务 | 为"QQ钱包"提供线上或线下展示空间 |
| 硬件要求 | 负责自身硬件平台搭建，确保系统安全性和稳定性 |
| 交易真实性 | 确保手机QQ钱包支付交易基于真实的交易背景 |
| 订单信息 | 保证订单信息的合法性、真实性、准确性和完整性 |
| 数据保留 | 保留交易数据和凭证至少五年 |
| 禁业行为 | 不得进行信用卡套现、洗钱、交易分单、虚假申请等违法经营行为 |

| 业务分离 | "实物+实名"、"机票与3C"、"虚拟产品"业务需分开申请商户号 |
| 巡检配合 | 配合本公司及相关外包服务机构的商户巡检工作 |

## 六、费用相关条款

**手续费**（第5.1条）：
- 手续费费率以商户平台核准记录的为准
- 收取方式：
  - （1）从每笔交易款中扣除
  - （2）从商户手续费专户中扣除
- 手续费调整：如遇政策调整，本公司有权调整费率，提前5个工作日书面通知

**发票**：
- 本公司每月开具上月增值税服务费发票
- 如需增值税专用发票，需提交一般纳税人资质材料

---

## 七、结算规则

**结算账户**（第5.3条）：
- 为商户申请本服务时提交的同名银行账户
- 变更结算账户需另行签订补充协议

**结算周期**：
- T日交易：按系统核准记录的期限划付至银行账户
- 接入服务不满90日或连续正常交易不满30日：不提供T+0资金结算
- 本公司有权根据经营需要调整结算周期和结算转账起点
- 对账数据以本公司数据为准

**保证金**（第6条）：
- 本公司有权要求缴纳风险保证金
- 可根据经营状况、业务变化及实际赔付情况调整保证金额度
- 注销本服务满一年无投诉或纠纷的，保证金无息归还
- 违约时，本公司有权直接使用保证金进行赔付

---

## 八、退款条款（第7条）

1. 商户账户需有足够退款的账存资金
2. 退款不再另行收取交易手续费（如银行另收费，由商户自行承担）
3. 商户负责解决用户拒付交易款项导致的退款纠纷

---

## 九、违约责任及违规处理

**违约情形**（第9条）：
- 直接或间接参与欺诈、出租出借出售账户、网上买卖POS机等
- 经营及财务状况恶化、被列入"严重违法失信企业名单"
- 假冒他人身份或虚构代理关系开立商户号
- 违规操作、经指出拒不纠正
- 進入破产程序、解散、营业执照被吊销
- 恶意欠款、实施有损本公司利益行为、利用服务从事非法业务
- 系统存在安全问题经通知后拒不整改
- 出现风险事件或存在交易异常

**处理措施**：
- 解除本协议
- 要求承担相关赔偿责任
- 信息报送至中国人民银行和公安部及相关监管机构

---

## 十、争议解决条款

- **适用法律**：中华人民共和国大陆地区法律
- **争议解决**（第16条）：友好协商，协商不成的，提交被告所在地有管辖权的人民法院诉讼解决

## 十一、其他重要条款

**协议变更**（第15条）：
- 本公司可能根据国家法律法规变化、市场发展等情况修改本协议
- 变更事项在商户平台公告30日后生效

**协议有效期**（第17条）：
- 有效期一年
- 届满前一个月内双方均未提出书面终止，视为自动延期一年

**保密条款**（第8条）：
- 未经书面同意，不得将协议内容、用户个人信息、支付信息等透露给第三方
- 保密义务在协议终止后仍然有效

**知识产权**（第14条）：
- 本公司提供的内容、软件系统、技术接口等知识产权归本公司所有

**不可抗力**（第11条）：
- 因不能预见、不能避免且不能克服的客观情况导致无法提供服务的，不承担责任

**金融消费者权益保护**（第13条）：
- 不得误导销售、强制搭售、重复收费
- 遵循合法、正当、必要原则处理金融消费者个人信息
- 如实充分披露与金融消费者有重大利害关系的内容

---

## 十二、禁业领域（第12.4条）

商户不得直接或变相从事：
- 互联网赌博、色情平台
- 互联网销售彩票平台
- 非法外汇、贵金属投资交易平台
- 非法证券期货类交易平台
- 代币发行融资及虚拟货币交易平台
- 未经批准通过互联网开展资产管理业务
- 未取得省级政府批文的大宗商品交易场所等非法交易
