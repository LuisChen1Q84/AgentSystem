## 一、核心说明

根据合规要求，现将相关行业、所需特殊资质及结算规则公示如下，供商家参考。但由于各个行业和商家的具体情况不尽相同，**不具有合同法律效力**，具体费率以商家在微信支付商户平台中的核准记录为准。

如发现商户经营内容包含需要特殊资质的业务，请根据审核提示补充相关资料。

---

## 二、结算规则字段说明

| 字段 | 说明 |
|------|------|
| **结算规则ID** | 结算规则的唯一标识 |
| **主体类型** | 个体户、企业等 |
| **结算规则描述** | 费率（0.2%-0.6%等）、入账周期（T+1等）、是否支持信用卡 |
| **适用行业范围概述** | 适用的行业场景描述 |
| **行业ID** | 行业分类编号 |
| **行业名称** | 具体行业类别（如服饰鞋包、美妆日化、健身运动服务、快递、物流等） |
| **特殊资质是否必填** | 是/否 |
| **特殊资质** | 具体需要的资质证明（如《快递业务经营许可证》、《金融许可证》等） |

## 四、特殊资质要求

### 快递行业
- 《快递业务经营许可证》

### 物流行业
- 《道路运输经营许可证》
- 网络货运需额外提供：《增值电信业务许可证》、《三级信息系统安全等级保护备案证明》

### 金融行业
- 《营业执照》含"消费金融"
- 《金融许可证》或地方银保监局批复文件

### 航空票务
- 《航空公司营业执照》或《航空公司机票代理资格证》
