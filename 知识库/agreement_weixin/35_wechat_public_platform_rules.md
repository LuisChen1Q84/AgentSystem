## 一、核心运营原则

1. **建立良好的用户体验**
   - 开发含有丰富交流与互动元素的公众号
   - 为用户提供更多选择和控制
   - 提供有价值的、持续性的内容

2. **要值得信赖**
   - 充分尊重用户并理解用户
   - 遵守国家相关法律法规
   - 不发送垃圾信息，不存在过度营销行为

---

## 二、禁止行为规则（违规内容分类）

### 2.1 注册规范相关禁止行为
- 使用虚假邮箱地址
- 提供虚假身份资料
- 账号名称涉及色情、暴力等违法违规内容
- 账号名称侵害他人名誉权、肖像权、知识产权、商业秘密
- 批量注册相似公众号
- 无正当理由使用与已有公众号重复的名称

### 2.2 认证规范禁止行为
- 使用与认证主体不紧密相关的信息命名
- 未经授权使用政府、企业、媒体、医院等机构名称
- 使用通用词汇、广义归纳词等不具识别性的词汇

### 2.3 行为规范禁止行为

| 违规类型 | 具体内容 |
|---------|---------|
| **使用外挂行为** | 未经许可使用插件、外挂、第三方工具 |
| **刷粉行为** | 利用第三方平台、外挂、漏洞进行刷粉 |
| **诱导行为** | 强制/诱导分享至朋友圈、诱导关注、违规导流、诱导点击 |
| **恶意篡改功能** | 篡改公众平台功能、交互页面或文案 |
| **浪费账号资源** | 连续210日未登录 |
| **滥用原创声明** | 整合引用内容、公共公开内容、营销宣传内容等 |
| **滥用赞赏功能** | 募捐、赌博、抽奖、售卖商品等 |
| **违法违规经营** | 未取得审批文件开展需要特许资质的业务 |
| **滥用模板消息** | 发送与场景不一致的广告营销类消息 |
| **滥用客服消息** | 发送无关消息骚扰用户 |
| **多级分销经营** | 类似传销的多级分销欺诈行为 |
| **欺诈/网赚行为** | 发布欺诈信息、虚假夸大宣传、刷单等 |
| **恶意违规及对抗** | 批量注册违规账号、买卖借用身份信息、规避审核 |
| **有偿投票行为** | 强制或吸引用户以有偿方式投票 |
| **滥用付费功能** | 付费内容与试读不符、非原创内容用于付费等 |
| **虚拟支付行为** | 在iOS系统上展示购买支付功能 |
| **强制关注行为** | 扫码点餐、开发票等强制关注 |
| **虚拟货币/数字藏品交易** | 发行、交易虚拟货币或提供数字藏品二级交易服务 |
| **混淆行为** | 恶意混淆腾讯官方功能或政府机构 |

### 2.4 发送内容禁止规范

| 类别 | 禁止内容 |
|-----|---------|
| **侵权类** | 侵犯企业名称、商标、肖像权、隐私权、知识产权、商业秘密 |
| **色俗类** | 淫秽色情内容、低俗配图、约炮经历描述 |
| **暴恐类** | 暴力伤害画面、虐待儿童、吸食毒品、自残 |
| **赌博类** | 组织赌博、传授赌博技巧、博彩活动 |
| **危害安全类** | 钓鱼网站、病毒、恶意代码 |
| **涉黑类** | 替人复仇、黑社会性质信息、非法集会 |
| **非法物品类** | 买卖发票、假烟、走私物品、违法办证 |
| **不实信息类** | 编造名人去世、虚假政策、食品安全谣言 |
| **骚扰类** | 诱导点击、营销内容骚扰 |
| **封建迷信类** | 算命、冥婚、看相、取名等 |
| **第三方商业营销** | 未标明"广告"、未经审查发布医疗广告 |

## 三、违规处理规则与处罚措施

### 3.1 处罚措施等级

| 处罚等级 | 具体措施 |
|---------|---------|
| **轻微** | 警告、删除内容、要求整改 |
| **中等** | 限制群发、限制关注、限制分享、限制搜索 |
| **严重** | 封禁相关能力、删除粉丝、清空数据 |
| **极严重** | 临时封号、永久封号、注销账号 |

### 3.2 阶梯式处罚机制

- **首次违规**: 警告、删除内容、限制部分功能
- **多次违规**: 限制或禁止使用部分或全部功能
- **严重违规**: 短期或永久封禁直至注销
- **涉及违法**: 永久封号，移交司法机关

### 3.3 特殊处罚规则

- **多级分销**: 限制功能至永久封号，拒绝再次提供服务
- **欺诈行为**: 永久封号处理
- **违法违规经营**: 责令限期整改，限制功能至永久封号
- **恶意违规及对抗**: 永久封号并拒绝提供注册/认证服务

## 四、数据使用规范

- 未经用户明确同意不得复制、存储、使用用户数据
- 不得收集、存储用户隐私信息
- 地理位置数据使用需获得用户同意

---

## 五、支付规范

- 不得实施非法吸收公众存款、集资诈骗等违法犯罪行为
- 不得禁止或阻碍用户使用微信支付

- 公众号使用风险由用户自行承担
