## 一、目的与原则

为促进微信支付商业健康、开放、有序发展，保障广大微信支付用户的合法权益，财付通支付科技有限公司制定本规则。本规则作为《微信支付服务协议》的附件，与主协议具有相同法律效力。

---

## 二、定义

| 术语 | 定义 |
|------|------|
| 商户 | 已与微信支付完成"主协议"签约的全部主体 |
| 整改警告 | 微信支付对商户不当行为进行提醒和告诫，要求商户在指定时限内完成自查及整改 |
| 调低支付限额 | 对异常交易商户的日支付限额进行调整 |
| 暂时冻结结算 | 对异常交易商户暂时冻结结算 |
| 暂时关闭支付 | 对异常交易商户暂时关闭收款功能 |
| 解除主协议 | 终止与商户的主协议关系 |

---

## 三、违规行为及处罚

| 违规类型 | 行为特征 | 处罚措施 |
|----------|----------|----------|
| **赌博** | 为参与赌博者提供赌场、赌具从中收取费用；或以金钱/有价值物品作为赌注营利 | 关闭支付权限 + 冻结结算 + 加入黑名单 |
| **淫秽色情** | 提供有毒害的文字/图片/音频/视频等信息内容 | 关闭支付权限 + 冻结结算 + 加入黑名单 |
| **一元购** | 商品按等份出售，全部售出后计算中奖号码 | 关闭支付权限 + 冻结结算 + 加入黑名单 |
| **纯返利** | 以佣金返现吸引用户，用新用户资金支付旧用户返利，形成层压式资金链条 | 整改期延长结算至T+7，整改后改为T+3，观察30日后决定是否恢复T+1；性质恶劣者立即解除协议 |
| **多级分销返利** | 以加入资格费或购买商品获得加入资格，按发展人员数量计酬 | 同上 |
| **信用卡套现** | 虚构交易、虚开价格、现金退款等方式套取现金 | 整改、关闭信用卡支付渠道；严重者解除协议 |
| **商户欺诈** | 收款后未提供商品/服务且拒不退还款项 | 延长结算周期，要求限期处理投诉；性质恶劣者立即解除协议 |
| **二清** | 以平台对接或"大商户"模式留存用户结算资金并自行清算 | 警告、整改；严重者解除协议 |
| **虚假交易** | 虚构或隐瞒交易事实获取虚假交易量 | 整改、延长结算周期；严重者解除协议 |
| **诱导支付** | 通过不正当途径诱导、欺骗用户付费 | 处理投诉，关闭出资金能力，逾期未处理则关闭支付权限 |
| **违规使用支付服务接口** | 将接口提供给无关联第三方使用；添加无资金管理关系的账户 | 停止服务、限期改正；逾期未改正则解除协议 |
| **跨类目经营** | 实际经营内容与类目不符；违规使用低费率通道 | 调整限额/结算周期/费率；需特殊资质者逾期未补充则解除协议 |
| **违反协议约定** | 未在表中列明的违约行为 | 整改、延长结算周期；严重者解除协议 |

## 四、规则公布/修改

- 微信支付有权随时对本规则进行单方面变更
- 以公告方式公布，无需单独通知商户
- 商户继续使用服务即视为同意变更后的规范
- 本规则自公布之日起生效
