## 第一条 定义（1.1-1.10）

| 术语 | 定义 |
|------|------|
| **微信** | 腾讯提供的跨平台通讯工具，包含关系链拓展工具、微信公众帐号、开放平台等 |
| **微信公众平台** | 腾讯提供的互联网技术服务平台 |
| **微信公众帐号** | 商户在微信公众平台注册的登录帐号 |
| **微信支付服务** | 财付通依托微信及微信公众平台提供的货币资金转移服务 |
| **用户** | 使用微信支付服务向商户购买商品或服务的自然人 |
| **微信支付商户号** | 财付通为商户配置的电子簿记，用于记载身份信息、交易信息、资金余额 |
| **商户平台** | 财付通提供的商户管理软件系统 |
| **商户平台登录账号** | 登录商户平台的账号 |
| **结算** | 财付通将交易资金记账至商户号并提现至银行账户的行为 |

## 第二条 服务内容（2.1-2.2）

### 2.1 服务场景和功能
1. **付款码支付**: 用户展示微信钱包内的"刷卡条码/二维码"给商户扫描
2. **Native支付**: 商户生成支付二维码，用户用微信"扫一扫"完成支付
3. **JSAPI支付**: 用户在微信中打开商户H5页面，通过JSAPI接口完成支付
4. **APP支付**: 商户在移动端APP中集成开放SDK调起微信支付
5. **H5支付**: 在微信客户端外的移动端网页使用微信支付
6. **小程序支付**: 专门在小程序中使用的支付产品

### 2.2 服务限制
- 仅能用于商户在商户平台申报并经核准的商品或服务
- 每笔订单不能超过财付通规定的限额

## 第三条 财付通的权利和义务（3.1-3.12）

| 条款 | 内容 |
|------|------|
| 3.1 | 负责支付系统的建设、运行和管理，确保系统安全性 |
| 3.2 | 设立网上交易查询系统及查询接口 |
| 3.3 | 提供"用户证书"服务 |
| 3.4 | 依据登录账号、密码、用户证书校验支付指令真实性 |
| 3.5 | 仅提供货币资金转移服务，交易纠纷由商户自行处理 |
| 3.6 | 有权定期审核商户业务类型，违规者有权暂停或终止服务 |
| 3.7 | 无交易记录超过6个月或180日无交易额，有权暂停服务 |
| 3.8 | 发现可疑交易时有权延迟结算、延迟转账 |

## 第四条 商户的权利和义务（4.1-4.22）

### 4.1 账号开立和身份验证
- 需如实提供经营资质资料（营业执照、税务登记证、组织机构代码证、身份证明等）
- 信息变更需提前通知财付通

### 4.3 禁止存储用户敏感信息
- 不得存储用户银行账户密码、银行卡磁道信息、验证码等敏感信息

### 4.4 账号保管和挂失
- 妥善保管各账号、密码、手机短信验证码、API Key、API证书等

### 4.7 商户系统安全与维护
- 严格按照接入技术规范建设和运行计算机系统

### 4.8-4.9 真实交易背景与订单信息
- 承诺发起真实交易，不得利用服务从事虚假交易
- 交易数据和凭证至少保留三年

### 4.12-4.13 反套现，反洗钱
- 不得协助持卡人进行信用卡套现、洗钱、交易分单等违法经营行为

### 4.19 保障消费者合法权益
- 确保金融消费者权益保护程度符合法律法规要求
- 不得对不同支付方式的用户采取歧视性措施
- 遵循合法、正当、必要原则处理消费者个人信息

---

## 第五条 服务费用和交易结算（5.1-5.3）

### 5.1 微信支付手续费
- 商户使用本服务需缴纳支付手续费
- 费率以商户平台核准记录为准

### 5.3 交易结算
- 扣除手续费后剩余款项记账至商户号
- 提现账户为申请时提交的同名银行账户
- T+0提现服务需接入本服务满90日且连续正常交易满30日

## 第七条 退款条款（7.1-7.3）

- 商户发起退款需确保商户号中有足够资金
- 退款不再另行收取交易手续费（如银行收取费用由商户承担）
- 商户负责解决用户拒付导致的退款纠纷

- 未经书面同意，不得向第三方透露协议内容、用户个人信息、支付信息等
- 保密义务在本协议终止后五年内有效

## 第九条 违约责任（9.1-9.2）

### 9.2 商户违约情形
财付通有权单方面解除协议并要求赔偿：
- 参与欺诈、出租出借出售银行账户或支付账户
- 经营及财务状况恶化、被列入严重违法失信企业名单
- 法定代表人或负责人被列入黑名单
- 违规操作经指出拒不改正
- 无理拒绝或拖延交易查询、调单查询
- 进入破产程序、解散、营业执照被吊销
- 实施有损财付通或腾讯利益的行为
- 利用服务从事非法业务

财付通有权注销商户微信支付商户号的情形：

- 协议解除或终止

- 违反本协议或其他协议
