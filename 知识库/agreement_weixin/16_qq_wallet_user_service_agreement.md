## 一、QQ 钱包支付服务

### 1.1 服务定义
QQ 钱包支付服务是本公司依托 QQ 客户端为收付款人之间提供的货币资金转移服务，包括余额支付、快捷支付及其他支付服务。

### 1.2 余额支付
- 使用财付通支付账户中的余额进行资金转移
- QQ 号码与财付通账户相关联
- 可用于消费和转账
- 限额取决于实名认证情况

### 1.3 快捷支付
- 委托本公司直接对银行账户进行资金扣划和归集
- 需输入真实姓名、身份证件号码、银行卡卡号、有效期、银行预留手机号等信息
- 需经本公司与开户银行校验通过后方能使用
- 交易限额由本公司根据监管要求、交易风险等因素设定/调整

### 1.4-1.7 其他条款
- 授权本公司将信息与外部验证渠道核验
- "余额"资金为预付价值，不受《存款保险条例》保护
- 本公司有权修改或变更本协议内容，最新协议公告30日后生效

## 二、账户和信息安全

### 2.1 账户设置与信息管理
- 需设置 QQ 号登录密码和 QQ 钱包支付密码
- 需提交姓名、身份证号、手机号、银行卡信息等
- 需配合身份验证的情形：
  - 单笔交易 1 万元以上
  - 30天内累计 5 万元以上
  - 账户余额连续 10 天超过 5000 元
  - 买卖金融产品或服务
  - 身份信息存在疑点

### 2.2 信息保管要求
需妥善保管：
- 银行卡及其密码、CVV 码、有效期
- QQ 号码及其登录密码
- 支付密码、手势密码
- 手机等移动终端及 SIM 卡信息
- 生物识别信息（指纹、虹膜、人脸、声纹）

### 2.3 风险处理
- 发生信息泄露或遗失时需及时联系客服：95017
- 未及时联系本公司导致的损失由用户自行承担

## 三、本服务的功能

| 功能 | 说明 |
|------|------|
| 充值和提现 | 银行账户与 QQ 钱包余额之间的资金划转 |
| 消费支付 | 向商户购买商品和服务，支持二维码/条码免密支付 |
| 转账/收款 | 通过"付款"、"收款"、"红包"、"群收钱"等方式 |
| 免密支付 | 开通后在一定额度内无需校验密码即可完成付款 |
| 查询 | 可在手机 QQ 客户端查询交易记录 |
| 添加/解绑银行卡 | 关联或解绑银行卡 |
| 修改/重置密码 | 修改或重置支付密码 |
| 退款 | 退款可退回原银行卡或 QQ 钱包余额 |

## 四、本服务的使用

### 4.1 支付指令
- QQ 号在登录状态下发出的支付指令视为本人指令
- 支付指令一旦发出即立即生效

### 4.2 免密支付
- 一定额度、次数内的支付无需验证支付密码和手机短信动态码

### 4.3 指纹支付/面容支付
- 需手机支持指纹支付功能
- 手机未进行非官方破解/ROOT
- 需通过财付通实名认证
- 遵循"一纹一机"规则

### 4.4-4.17 其他使用规则
- 非即时到账服务，资金流转需要合理时间
- 6个月无交易记录可暂停服务
- 需仔细确认交易及其金额后进行支付
- 需达到实名认证标准
- 转账日限额和笔数根据风险管理情况设置
- 禁止用于非法目的

### 4.14 禁止行为
- 反对宪法基本原则、危害国家安全
- 侵害他人名誉、隐私等权利
- 冒用他人名义使用本服务
- 从事不法交易（贩卖枪支、毒品、违禁物）
- 赌博、传销、洗钱、套现
- 出租、出借、出售、购买银行账户或支付账户
- 假冒他人身份开立账户
