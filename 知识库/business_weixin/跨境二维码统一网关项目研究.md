# 跨境二维码统一网关（跨境二维码互联互通）项目研究

> 更新时间：2026-02-19
> 来源：SerpAPI 搜索整理

---

## 一、项目概述

### 1.1 项目名称

- **中文名称**：跨境二维码统一网关 / 跨境二维码互联互通
- **英文名称**：Cross-border QR Code Unified Gateway / Cross-border Payment Gateway (CPG)
- **简称**：CPG

### 1.2 主管单位

- **中国人民银行**（央行）
- **中国支付清算协会**

### 1.3 项目背景

为贯彻落实国务院关于提升境外来华人员支付服务便利化的决策部署，人行组织建立了跨境二维码统一网关，旨在打通境内外支付体系，让外国游客可以直接使用境外电子钱包在中国扫码支付。

## 三、核心参与方

### 3.1 境内机构

| 机构 | 角色 |
|------|------|
| **中国人民银行** | 主管单位、政策制定 |
| **中国支付清算协会** | 行业协调、标准制定 |
| **支付宝** | 境内受理机构 |
| **财付通（微信支付）** | 境内受理机构 |
| **网联清算有限公司** | **清算基础设施（关键）**、二维码业务清算 |
| **中国银联 / 银联国际** | 卡组织、跨境受理网络 |

#### 网联在CPG中的角色

| 功能 | 说明 |
|------|------|
| **清算职能** | 负责境内二维码支付交易清算 |
| **转接职能** | 连接境内支付机构与境外网关 |
| **监管职能** | 交易监测、备付金管理 |
| **技术支撑** | 系统对接、标准统一 |

### 3.2 竞合关系分析

#### 财付通（微信支付）VS 银联国际

| 维度 | 财付通（微信支付） | 银联国际 |
|------|---------------------|----------|
| **优势** | 用户基数大、扫码支付场景丰富 | 卡组织背景、境外受理网络广泛 |
| **劣势** | 境外用户基础薄弱 | 二维码受理场景有限 |
| **合作** | 共同接入CPG系统 | 提供境外卡受理网络 |
| **竞争** | 境外钱包扫码受理 | 跨境卡支付市场份额 |

#### CPG架构下的角色分工

```
CPG（跨境二维码统一网关）
├── 财付通/微信支付 → 境内二维码受理
├── 支付宝 → 境内二维码受理
├── 网联清算 → 境内清算、转接
└── 银联国际 → 境外卡受理、ATM网络
```

#### 网联与财付通的关系

| 维度 | 说明 |
|------|------|
| **清算关系** | 财付通通过网联进行二维码业务清算 |
| **合规监管** | 网联对财付通交易进行监测 |
| **断直连** | 财付通所有条码支付业务通过网联处理 |
| **CPG对接** | 网联是财付通接入CPG的关键通道 |

#### 市场影响

| 业务场景 | 受影响方 | 影响分析 |
|----------|----------|----------|
| 境外用户扫码 | 财付通、支付宝 | 新增境外用户流量 |
| 境外卡刷卡 | 银联国际 | 面临扫码分流压力 |
| 跨境汇款 | 银联、财付通 | 竞争加剧 |
| 境外来华消费 | 全支付机构 | 蛋糕做大，竞争合作并存 |

### 3.2 境外机构

| 机构 | 国家/地区 |
|------|-----------|
| **TenPay Global（钱宝科技）** | 财付通境外子公司 |
| **国际电子钱包** | 境外合作钱包（陆续接入中） |

## 四、项目架构

### 4.1 CPG（Cross-border Payment Gateway）

CPG是经中国人民银行批准的跨境支付基础设施，负责：
- 境外电子钱包与境内支付网络的对接
- 跨境交易清算
- 外汇兑换处理

### 4.2 互联互通模式

```
境外用户钱包 → CPG（跨境网关）→ 境内支付网络（微信支付/支付宝）→ 商户
```

## 五、对微信支付/财付通的影响

### 5.1 业务机会

| 业务类型 | 影响 | 优先级 |
|----------|------|--------|
| **境外用户扫码** | 新增境外用户群体 | 高 |
| **跨境消费** | 境外来华人员消费 | 高 |
| **商户受理** | 境外钱包扫码受理 | 高 |

### 5.2 挑战与风险

| 风险类型 | 具体内容 |
|----------|----------|
| **合规风险** | 外汇管理、反洗钱、KYC |
| **技术对接** | 多国钱包系统兼容 |
| **竞争格局** | 支付宝先发优势 |
| **费率压力** | 跨境交易费率博弈 |
