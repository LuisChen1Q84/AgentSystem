# 反洗钱非现场监管实用手册

**（内部资料，仅适用于北京地区）**
**版本号：V21-1**

---

**中国人民银行营业管理部反洗钱处**

本手册将反洗钱处工作中收到的常见非现场监管业务问题进行了梳理，供各义务机构反洗钱工作人员参考、查询。手册中的内容仅适用于北京地区，非北京地区的反洗钱监管业务，应以当地人民银行口径为准。本手册会根据实际需要不定期更新，如在操作中遇到疑问，请及时与反洗钱处工作人员联系。

# 第一部分 日常工作信息报告

## 1. 报告机构类别

### 1.1 向反洗钱处报告的机构类别

经营地址在北京地区的下列义务机构需向反洗钱处报送工作报告及其他信息资料：

- 开发性金融机构、政策性银行、商业银行、农村合作银行、农村信用社、村镇银行。
- 证券公司、期货公司、证券投资基金管理公司。
- 保险公司、保险资产管理公司。
- 非银行支付机构。
- 信托公司、金融资产管理公司、企业集团财务公司、金融租赁公司、汽车金融公司、消费金融公司、货币经纪公司、贷款公司、银行理财子公司。
- 互联网金融从业机构。
- 银行卡清算机构、资金清算中心、网络小额贷款公司以及从事汇兑业务、基金销售业务、保险代理和保险经纪业务的机构。
- 中国人民银行确定并公布的应当履行反洗钱和反恐怖融资义务的其他机构。

### 1.2 向反洗钱处报告的机构层级

- 所有法人机构（如银行总行、公司总部等）。
- 法人机构不在京的最高层级分支机构（例如，总行不在京的银行北京分行、总公司不在京的北京分公司等）。

对于归属同一总公司的多个在京平级分支机构，应指定一家机构作为牵头报告机构，或指定一名工作人员作为牵头报告人，向反洗钱处报送在京所有平级分支机构的反洗钱工作报告及其他信息资料。

### 1.3 法人机构注册地与经营地不一致

当法人机构注册地与经营地不一致时，相关反洗钱监管职责如下：

1. 反洗钱工作报告及其他信息资料——经营地人民银行。
2. 反洗钱数据报送主体资格申请——注册地人民银行。
3. 金融机构编码申请——注册地人民银行。

### 2.2 日常反洗钱工作信息的报告内容

报告内容按照义务机构的不同类型、行业有所区别。

**法人机构、商业银行分行、支付公司分公司的报告内容**

| 应报告的情况举例 | 时限要求 |
|------------------|----------|
| 1-1. 制定或者修订主要反洗钱内控制度 | 发生后10个工作日内 |
| 1-2. 牵头负责反洗钱工作的高级管理人员、牵头管理部门或者主要负责人调整 | 发生后10个工作日内 |
| 1-3. 本机构反洗钱岗位人员调整、联系方式变更（不含下级分支机构） | 发生后10个工作日内 |
| 1-4. 涉及本机构反洗钱工作的重大风险事项 | 发生后10个工作日内 |
| 1-5. 境外分支机构和控股附属机构受到当地监管当局或者司法部门开展的与反洗钱和反恐怖融资相关的执法检查、行政处罚、刑事调查或者发生其他重大风险事件 | 发生后10个工作日内 |
| 1-6. 洗钱风险自评估相关情况 | 董事会或高级管理层审定之日起10个工作日内 |
| 1-7. 内部检查、审计等工作开展情况 | 专项工作完成后，或持续性工作取得阶段成果 |
| 1-8. 宣传、培训等工作开展情况 | 专项工作完成后，或持续性工作取得阶段成果 |
| 1-9. 反洗钱工作有关的信息系统开发、改造情况 | 专项工作完成后，或持续性工作取得阶段成果 |
| 1-10. 调研、信息 | 撰写完成后 |

> **注：** 事项1-5，原则上要求在发生之日起10个工作日内报告。如有特殊情况无法按照规定时限报告，请于发生之日起10个工作日内与反洗钱处日常监管人员联系。

**其他行业分支机构的报告内容**

| 应报告的情况举例 | 时限要求 |
|------------------|----------|
| 2-1. 主要反洗钱内控制度修订（操作流程类制度不包含在内） | 发生后10个工作日内 |
| 2-2. 反洗钱工作机构和岗位人员调整、联系方式变更 | 发生后10个工作日内 |
| 2-3. 涉及本机构反洗钱工作的重大风险事项 | 发生后10个工作日内 |

**归属同一总公司的多个在京平级分支机构报告时：**

如各分支机构均执行总公司的统一内控制度，或开展了由总公司要求的相同性质工作，原则上仅需在总公司修订相关制度后上报一份内控制度，完成有关工作后统一上报一份工作报告，无需按照各分支机构分别上报工作信息。

对于报告事项2-2，只需报告牵头报告机构反洗钱工作机构和岗位人员调整、联系方式变更情况；其他平级分支机构岗位人员调整情况由牵头报告机构及牵头报告人自行掌握，无须报告。

机构经营地址变更、反洗钱岗位人员调整及联系方式变更时，除按规定完成报告，还应同步在反洗钱非现场监管系统中变更机构信息，具体方式参见本手册"3.5 在反洗钱非现场监管系统中变更机构信息"。

**通过反洗钱监管交互平台报告的注意事项**

通过反洗钱监管交互平台进行报备时，应选择"年度报告"、"即时报告"、"自评报告"、"洗钱类型分析报告"类别。除有特殊要求外，请不要使用"点对点信息发布"进行报备。

**通过反洗钱非现场监管系统报告的注意事项**

通过反洗钱非现场监管系统进行报备时，请参考以下内容选择文档类别：

- **组织架构**——反洗钱领导小组、牵头部门信息变更。
- **工作计划**——年度工作计划等。
- **内控制度**——内部制度、操作规程、指标模型等。
- **会议反馈**——召开董事会、反洗钱工作会等内部会议、参加人民银行组织的会议和培训后的反馈信息等。
- **联系人变更**——联系人、机构信息等变更。
- **宣传培训**——本机构开展的宣传、培训等工作。
- **审计报告**——反洗钱相关的审计、内部检查等工作。
- **工作总结**——年度工作总结，上述内容未涵盖的其他相关工作总结和工作报告，洗钱和恐怖融资风险自评估报告、业务/产品洗钱和恐怖融资风险评估报告等。
- **工作信息**——自主开展的调研、向人民银行反馈的问卷等。
- **其他**——所有不便分类的文件资料。

**通过寄送方式报告的注意事项**

未开通非现场监管系统的机构，请将报备文件刻盘，并通过邮寄方式向反洗钱处报告。

### 2.4 提交的报告或文件的抬头

提交对象为反洗钱处时，抬头应为："中国人民银行营业管理部反洗钱处"。同时，义务机构应根据实际情况决定是否起草正式文件（文件有红头和文件编号）。

提交对象为营管部时，抬头应为："中国人民银行营业管理部"。同时，义务机构应起草正式文件。

义务机构通过人行营管部办公室提交正式文件时，请同时在反洗钱监管交互平台或反洗钱非现场监管系统进行报备，以免反洗钱处在统计工作中出现遗漏。

## 3. 机构设立和注销

### 3.1 新设立机构应向反洗钱处提交的材料及报告途径

新设机构应提交以下材料：

1. 营业执照和经营许可证复印件，加盖公章。（扫描件）

## 2. 反洗钱数据报送系统

### 2.1 反洗钱数据报送系统（试报送）的功能

报告机构通过反洗钱数据报送系统（试报送）可以进行主体资格申请、数字证书申请、密码重置申请、大额交易试报告和可疑交易试报告工作。

### 2.2 反洗钱数据报送系统（二代系统）的功能

报告机构通过反洗钱数据报送系统可以进行主体资格信息变更、密码重置申请、大额交易报告、可疑交易报告、更新联系人信息、更新机构网点代码、上传和更新可疑交易监测标准对照表、查看通知公告等工作。

### 2.3 经营地在京机构的主体资格管理权归属

中国反洗钱监测分析中心负责报告机构的用户创建和信息变更，人行营业管理部反洗钱处负责注册地在京的报告机构管理工作。如果某机构注册地在北京，但是经营地不在北京，仍由人行营管部反洗钱处对该机构的主体资格进行管理。

### 2.4 需要接入反洗钱数据报送系统的机构类别

具有反洗钱报送主体资格的机构都应当接入反洗钱数据报送系统（试报送）和反洗钱数据报送系统（二代系统），包括法人机构（商业银行总行、其他类别机构的总公司等）和主报告机构。

> **注：** 此处的主报告机构指：当某些外资金融机构在中国境内没有设立法人机构，但存在多个分支机构时，需要指定某一个分支机构作为反洗钱数据主报告机构，负责报送其总部所属的全部分支机构发生的大额交易报告和可疑交易报告。此处的主报告机构与日常工作信息报告中的牵头报告机构不是同一个范畴（第一部分 日常工作信息报告 1.2 中的内容）。

### 2.5 接入渠道和技术准备

**接入渠道**

1. 银行业报告机构、银行卡清算机构和资金清算中心通过金融城域网登录二代系统，并需要使用数据证书进行电子身份认证。
2. 其他报告机构通过互联网登录二代系统，并需要使用数字证书或动态口令进行电子身份认证。

**接入的技术准备**

反洗钱数据报送系统的接入方法请参见《中国人民银行办公厅关于做好反洗钱监测分析二代系统上线工作的通知》（银办发〔2018〕213号）、《人民银行反洗钱监测分析二代系统操作手册》（报告机构版）。

### 2.6 接入流程

### 2.7 操作手册获取方法

### 2.8 变更机构信息

## 3. 反洗钱非现场监管系统

### 3.1 系统的功能

为落实反洗钱"风险为本"监管原则，中国人民银行营业管理部开发建设了反洗钱非现场监管系统，于2019年9月2日正式上线。目前，该系统的主要功能有机构管理、资料报备（文件报备时请选择对应的文档类别，参见1.7（2）的具体内容）、风险评估和分类评级。

### 3.2 系统的接入对象

需要接入反洗钱非现场监管系统的机构包括：

- 法人机构（商业银行总行、金融机构总公司）
- 在京的第一层级分支机构（商业银行北京分行、金融机构北京分公司）
- 保险集团

对于归属同一总公司的多个在京平级分支机构，原则上在非现场监管系统仅设立一个用户，进行日常工作信息报备（具体要求，详见第一部分 日常工作信息报告 1.2 中的内容）。

### 3.3 系统的接入流程

申请加入反洗钱非现场监管系统的机构，首先应填写《反洗钱非现场监管系统用户申请（变更）表》（附件2），由机构负责人或主管反洗钱工作的高级管理人员签字并加盖公章。然后将《反洗钱非现场监管系统用户申请（变更）表》的扫描件通过互联网发送至fxqc@bj.pbc.gov.cn。

接入反洗钱非现场监管系统后，请在"系统管理"模块，"机构信息"一栏，及时维护机构信息有关内容。

### 3.4 使用手册的获取

需要反洗钱非现场监管系统使用手册的机构，可以登录反洗钱处的公共邮箱pbcaml_bj@126.com（登录密码pbcaml）进行下载。（注：pbcaml中"l"为小写字母"l"）

### 3.5 变更机构信息

- **机构名称变更**

  机构名称变更后，首先应填写《反洗钱非现场监管系统用户申请（变更）表》（附件2），由机构负责人签字并加盖公章。然后将《反洗钱非现场监管系统用户申请（变更）表》的扫描件通过反洗钱非现场监管系统上传。

  请注意，机构名称变更申请处理完成后，反洗钱非现场监管系统的登录用户名仍为变更前的机构名称（原名称），登录系统后显示的机构名称信息为变更后的机构名称（新名称）。

- **营业执照注册地址变更**

  必须自行在非现场监管系统中进行修改，具体方式参见"3.3 反洗钱非现场监管系统的接入流程"中维护机构信息相关内容。

- **经营地址变更**

  可自行在非现场监管系统中进行修改，具体方式参见"3.3 反洗钱非现场监管系统的接入流程"中维护机构信息相关内容。

- **联系人变更**

  可自行在非现场监管系统中进行修改，具体方式参见"3.3 反洗钱非现场监管系统的接入流程"中维护机构信息相关内容。

- **牵头部门负责人变更**

  可自行在非现场监管系统中进行修改，具体方式参见"3.3 反洗钱非现场监管系统的接入流程"中维护机构信息相关内容。

- **机构注销**

  机构注销后，应在反洗钱非现场监管系统中提交《反洗钱非现场监管系统用户注销申请》，并加盖公章。此撤销申请没有模板，将事由、联系人和联系方式注明即可。

### 3.6 用户登录密码重置

应填写《反洗钱非现场监管系统密码重置申请表》（附件3），经机构负责人和/或反洗钱部门负责人签字后，将申请表的扫描件发至fxqc@bj.pbc.gov.cn。

## 4. 反洗钱监管交互平台

### 4.1 平台的功能

反洗钱监管交互平台（新版）系统是中国人民银行反洗钱局2019年开发建设的，分别搭建在金融城域网平台和互联网平台。该系统用于机构上报材料、监管部门与义务机构进行信息交互等。

### 4.2 平台的接入对象

根据《中国人民银行办公厅关于开展反洗钱监管交互平台系统推广工作的通知》（银办发〔2020〕115号）有关工作要求，接入反洗钱监管交互平台的机构为法人机构、由中国人民银行营业管理部反洗钱处确定的在京分支机构。

### 4.3 平台的接入流程

**业务网交互平台**

1. 义务机构申请使用业务网交互平台，应填写《反洗钱监管交互平台系统义务机构使用登记表》（附件4）、《反洗钱监管交互平台系统义务机构用户操作申请表》（附件5），原则上每个义务机构申请一个用户。
2. 应按照《中国人民银行金融城域网入网管理办法》（银办发〔2013〕151号文印发）的工作要求，接入或确认已接入金融城域网。

**互联网交互平台**

1. 义务机构申请使用互联网交互平台，应填写《反洗钱监管交互平台系统义务机构使用登记表》（附件4）、《反洗钱监管交互平台系统义务机构用户操作申请表》（附件5）、《义务机构数字证书操作申请表》（附件6），原则上每个义务机构申请一个用户。
2. 待收到数字证书服务平台短信通知后，通过短信验证码的方式登录数字证书服务平台，并根据数字证书服务平台提供的操作手册填写数字证书申领信息，完成数字证书申领。

**申请表提交途径**

上述申请表应由机构反洗钱主管领导签字或加盖公章，并将扫描版及可编辑版通过反洗钱非现场监管系统提交。对于没有开通反洗钱非现场监管系统的机构，应发送表格至电子邮箱fxqc@bj.pbc.gov.cn。

### 4.4 用户名的变更

交互平台用户因岗位变动等原因调离反洗钱岗位的，应向反洗钱处提交两份《反洗钱监管交互平台系统义务机构使用登记表》（附件4），一份为原用户停用申请，需勾选"用户停用"；另一份为建立新用户申请，需勾选"用户申请"。上述申请表应由相关负责人签字或加盖公章，并将扫描版及可编辑版发送至电子邮箱fxqc@bj.pbc.gov.cn。

### 4.5 用户登录密码的重置

用户因登录密码遗失无法正常登录交互平台的，应向反洗处提交《反洗钱监管交互平台系统义务机构用户操作申请表》（附件5）。申请表中需勾选"密码重置"，由相关负责人签字或加盖公章，并将扫描版及可编辑版发送至电子邮箱fxqc@bj.pbc.gov.cn。

### 4.6 用户登录密码的解锁

用户因多次输入错误密码导致用户被锁定、无法正常登录交互平台的，应向反洗钱处提交《反洗钱监管交互平台系统义务机构用户操作申请表》（附件5）。申请表需勾选"用户解锁"，由机相关负责人签字或加盖公章，并将扫描版及可编辑版发送至电子邮箱fxqc@bj.pbc.gov.cn。

### 4.7 数字证书使用人、手机号码变更

当互联网交互平台使用人、使用人手机号码发生变更时，应填写《义务机构数字证书操作申请表》（附件6）。申请表中需勾选"变更手机号码"，由相关负责人签字或加盖公章，并将扫描版及可编辑版发送至电子邮箱fxqc@bj.pbc.gov.cn。

### 4.8 平台的停用

**业务网交互平台**

义务机构需要停止使用业务网交互平台时，应及时填写《反洗钱监管交互平台系统义务机构用户操作申请表》（附件5）。申请表需勾选"用户停用"，于备注栏写明停用原因，由相关负责人签字或加盖公章，并将扫描版及可编辑版发送至电子邮箱fxqc@bj.pbc.gov.cn。

**互联网交互平台**

义务机构需要停止使用互联网交互平台时，应及时填写《反洗钱监管交互平台系统义务机构用户操作申请表》（附件5），申请表需勾选"用户停用"，于备注栏写明停用原因。以及《义务机构数字证书操作申请表》（附件6），申请表勾选"注销"。申请表由相关负责人签字或加盖公章，并将扫描版及可编辑版发送至电子邮箱fxqc@bj.pbc.gov.cn。
