# 关于与银行开展反诈联防联控合作的情况介绍

## 一、合作背景

当前，电信网络诈骗形势严峻，诈骗手段不断翻新，给人民群众财产安全带来严重威胁。公安机关在打击治理电信网络诈骗工作中，需要与银行等金融机构紧密配合，建立联防联控机制，形成打击合力。

## 二、合作目标

1. **建立信息共享机制**：实现公安机关与银行之间的诈骗信息实时共享，及时阻断诈骗资金流转通道。
2. **强化账户管理**：加强对涉诈账户的识别和管控，从源头上防范诈骗风险。
3. **开展预警劝阻**：银行协助公安机关对潜在受害人进行预警和劝阻，有效阻止案件发生。
4. **优化追赃挽损**：建立快速冻结、扣划机制，最大程度挽回受害人损失。

## 三、合作内容

### （一）信息共享与查询

- 建立反诈专线查询通道，银行实时响应公安机关的账户查询请求
- 银行定期向公安机关报送涉诈账户线索
- 共同建设反诈信息数据库，实现数据互联互通

### （二）快速冻结机制

- 开通涉诈账户快速冻结绿色通道，实现"秒级"响应
- 建立7x24小时应急联络机制，确保随时启动冻结程序
- 完善冻结资金返还流程，提高追赃挽损效率

### （三）风险防控合作

- 银行对异常交易进行实时监测，及时向公安机关报告可疑账户
- 联合开展风险账户排查行动，清理整顿涉诈账户
- 建立客户身份识别机制，严格落实实名制要求

### （四）宣传教育合作

- 共同开展反诈宣传活动，提高群众防范意识
- 银行在营业网点设置反诈宣传阵地
- 配合公安机关开展反诈进校园、进社区活动

## 四、合作成效

1. **打击效能显著提升**：通过快速冻结机制，成功阻断大量诈骗资金转移
2. **预警劝阻成效明显**：协助劝阻大量潜在受害人，避免财产损失
3. **账户治理取得实效**：清理了大量涉诈账户，净化了金融环境
4. **群众满意度提高**：追赃挽损工作得到群众认可

## 五、下一步工作计划

1. 深化信息共享机制，进一步提升数据交互效率
2. 扩大合作银行范围，实现更广泛的联防联控覆盖
3. 加强技术对接，提升智能化风险防控能力
4. 完善工作机制，推动合作向纵深发展

## 六、典型案例

### 案例一：快速冻结成功挽损

某日，公安机关接到群众报警称遭遇冒充公检法诈骗。接报后，公安机关立即启动快速冻结机制，通过银行查询并冻结涉诈账户，成功在资金转出前将其冻结，为受害人挽回损失50万元。

### 案例二：精准预警成功劝阻

银行在日常监测中发现一客户账户交易异常，短时间内有多笔大额转入转出，疑似被诈骗分子利用。银行立即启动预警机制，联系公安机关。公安机关迅速开展劝阻工作，成功避免该群众被骗30万元。
