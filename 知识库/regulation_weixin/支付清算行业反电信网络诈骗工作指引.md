# 支付清算行业反电信网络诈骗工作指引

---

## 第一章 总则

**第一条** 为深入贯彻落实党中央、国务院关于打击治理电信网络诈骗违法犯罪工作的决策部署，加强支付清算行业反电信网络诈骗工作，强化行业风险管理职责，维护社会公众合法权益，依据《中华人民共和国网络安全法》《中华人民共和国反电信网络诈骗法》《非银行支付机构网络支付业务管理办法》等法律法规和行业规范，制定本指引。

**第二条** 本指引适用于在中国境内开展支付清算业务的银行业金融机构、非银行支付机构、清算机构等（以下统称"支付清算机构"）。

**第三条** 支付清算行业反电信网络诈骗工作应当坚持以下原则：

| 原则 | 说明 |
|------|------|
| 坚持党的领导 | 贯彻落实党和国家关于打击治理电信网络诈骗违法犯罪工作的方针政策 |
| 坚持以人民为中心 | 切实保护人民群众财产安全 |
| 坚持依法合规 | 严格遵守法律法规和行业规范 |
| 坚持风险为本 | 建立健全风险防控体系 |
| 坚持协同联动 | 形成行业反诈工作合力 |

## 第四章 可疑交易识别

### 第十七条 可疑交易识别应当重点关注以下情形

#### （一）账户异常特征

| 特征类型 | 具体表现 |
|----------|----------|
| 开户时间异常 | 短期内集中开立多个账户，或在异常时间（深夜、节假日）开立账户 |
| 账户类型与客户身份不符 | 客户身份、职业、年龄等与账户类型明显不匹配 |
| 账户使用异常 | 长期不活跃账户突然出现大额资金流动，或频繁快进快出 |
| 联系信息异常 | 多个账户使用相同或相似的联系电话、地址、IP地址等 |
| 客户身份信息异常 | 提供虚假身份信息，或冒用他人身份开立账户 |

#### （二）交易异常特征

| 特征类型 | 具体表现 |
|----------|----------|
| 交易金额异常 | 单笔或累计交易金额明显超出客户正常交易水平 |
| 交易频率异常 | 交易频率明显高于正常水平，或短时间内出现大量交易 |
| 交易时间异常 | 在非正常时间（深夜、凌晨）出现频繁交易 |
| 交易对手异常 | 与身份不明确或涉嫌诈骗的账户进行交易 |
| 资金流向异常 | 资金快速转入转出，或资金流向与客户身份、职业明显不符 |
| 分散转入集中转出 | 资金分散转入后集中转出，或采取分层转移方式转移资金 |
| 过渡资金特征 | 账户资金留存时间短，主要用于过渡资金周转 |

#### （三）行为异常特征

| 特征类型 | 具体表现 |
|----------|----------|
| 客户行为异常 | 客户对账户交易情况漠不关心，或对询问表现不耐烦 |
| 业务办理异常 | 客户办理业务时表现紧张、焦虑，或有专人陪同、指点 |
| 账户管理异常 | 客户对账户进行异常操作，如频繁变更绑定手机号码、邮箱等 |
| 配合度异常 | 客户不配合身份核验，或拒绝提供必要的交易证明材料 |

### 可疑交易分级处理

| 可疑程度 | 管控措施 |
|----------|----------|
| **一般可疑** | 加强关注，继续监测，必要时与客户进行核实 |
| **较强可疑** | 采取限制非柜面交易、降低交易限额等管控措施 |
| **高度可疑** | 暂停账户所有交易，及时向公安机关报告，并按规定采取账户管控措施 |

### 调查措施

| 信息核实 | 与相关机构进行信息核实 |
| 其他措施 | 其他必要的调查措施 |

### 配合公安机关工作

| 工作内容 | 要求 |
|----------|------|
| 查询涉案账户信息 | 严格按照法律程序和规定办理 |
| 冻结涉案账户资金 | 及时、准确地提供相关信息 |
| 扣划涉案账户资金 | 做好工作记录，保存相关证据材料 |
| 提供交易记录 | 对涉案账户信息严格保密 |

## 第八章 附则

### 名词解释

| 术语 | 定义 |
|------|------|
| **电信网络诈骗** | 以非法占有为目的，利用电信网络技术手段，通过虚构事实或者隐瞒真相的方法，骗取公私财物的行为 |
| **可疑交易** | 客户或账户的交易行为存在异常，可能涉嫌电信网络诈骗的交易 |
| **涉诈账户** | 涉嫌用于电信网络诈骗的银行账户和支付账户 |

## 附录一：涉诈可疑交易识别要点速查表

| 类别 | 识别要点 | 风险等级 |
|------|----------|----------|
| 开户环节 | 短期内集中开户、异常时间开户、身份信息存疑 | 高 |
| 账户使用 | 长期不动户突然启用、频繁挂失补卡、账户分拆 | 高 |
| 交易特征 | 快进快出、分散转入集中转出、过渡资金 | 高 |
| 交易对手 | 与涉诈名单账户交易、与异常账户交易 | 高 |
| 客户行为 | 不配合身份核验、拒绝说明交易用途 | 中 |
| 通讯特征 | 使用虚拟运营商号码、位置信息异常 | 中 |

## 附录二：涉诈账户管控措施指引

| 可疑程度 | 管控措施 | 报告要求 |
|----------|----------|----------|
| 一般可疑 | 加强监测，与客户核实 | 留存记录 |
| 较强可疑 | 限制非柜面交易，降低限额 | 报告分支机构和总部 |
| 高度可疑 | 暂停非柜面交易，报告公安机关 | 立即报告，必要时报案 |

## 附录三：主要电信网络诈骗类型识别

| 诈骗类型 | 说明 |
|----------|------|
| 冒充公检法诈骗 | 冒充公安机关、检察院、法院工作人员，以涉嫌洗钱、非法出入境等名义实施诈骗 |
| 冒充客服诈骗 | 冒充电商平台、银行、支付机构客服，以退款、账户异常等名义实施诈骗 |
| 网络刷单诈骗 | 以兼职刷单为名，先给予小利后实施大额诈骗 |
| 网络贷款诈骗 | 以低息、快速贷款为名，要求先缴纳手续费、保证金等实施诈骗 |
| 投资理财诈骗 | 以高收益、低风险投资为诱饵，实施虚假投资理财诈骗 |
| 冒充熟人诈骗 | 通过盗取QQ、微信等账号，冒充熟人实施诈骗 |
| 虚假中奖诈骗 | 以中奖为名，要求缴纳个人所得税等实施诈骗 |
| 冒充领导诈骗 | 冒充企业领导、政府官员，以紧急汇款等名义实施诈骗 |
